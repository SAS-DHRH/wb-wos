# WISE Image Search Notebook

File: [image-search.ipynb](./image_search.ipynb)

This notebook provides a hands-on introduction to [WISE](https://www.robots.ox.ac.uk/~vgg/software/wise/), an open-source image search engine developed by the [Visual Geometry Group (VGG)](https://www.robots.ox.ac.uk/~vgg/) at the University of Oxford. It allows you to search image collections using natural language descriptions.

## Requirements

A Google account and the Google Chrome browser (strongly recommended) are required to run this notebook, as it runs on [Google Colab](https://colab.research.google.com/) (or "Colaboratory") and loads images from Google Drive. No programming experience or local software installation is needed.

## Origin

This notebook was adapted from [VGG's original WISE notebook](https://github.com/ox-vgg/wise/blob/main/notebooks/image_search.ipynb) to support use of images from Google Drive. The details of the changes made are documented in the version history at the bottom of the notebook itself.

## A note on stability

WISE is under active development, which means future changes to the software could break this notebook. To reduce this risk, the notebook is pinned to a specific version of WISE rather than always fetching the latest. If you encounter unexpected errors, check the version history in the notebook and compare against any recent changes in [VGG's repository](https://gitlab.com/vgg/wise/wise).
